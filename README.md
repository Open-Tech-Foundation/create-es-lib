## Requirements

- Node.js ^12.22.0 || ^14.17.0 || >=16.0.0
- Latest Npm or Yarn

## Usage

```bash
# With npm
$ npx @open-tech-world/create-es-lib your-lib-name

# With yarn classic
$ yarn create @open-tech-world/es-lib your-lib-name

# With yarn berry
$ yarn dlx @open-tech-world/create-es-lib your-lib-name
```
